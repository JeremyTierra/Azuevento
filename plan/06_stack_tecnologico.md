# Stack Tecnológico

## Frontend
- React Native
- Arquitectura basada en componentes
- Navegación móvil (stack + tabs)

## Backend
- Spring Boot
- API REST
- Autenticación basada en tokens

## Base de Datos
- PostgreSQL
- Modelo relacional

## Tipo de aplicación
- Aplicación móvil multiplataforma (Android / iOS)

## Arquitectura
- Cliente móvil → API REST → Base de datos
